HTML Basics

HTML (HyperText Markup Language) is the standard language for creating web pages.

<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
  </body>
</html>

Key HTML Tags:

  • <h1> to <h6>: Defines headings. <h1> is the largest, <h6> is the smallest.
  • <p>: Defines a paragraph.
  • <a>: Defines a hyperlink. Example: <a href="https://example.com">Visit Example</a>
  • <img>: Embeds an image. Example: <img src="image.jpg" alt="Description">
  • <ul> and <li>: Creates an unordered list with list items.

Example: Adding an Image

<img src="https://via.placeholder.com/300" alt="Example Image">

HTML is easy to learn and is the foundation of web development. Experiment with different tags to create your own pages!

CSS Basics

CSS (Cascading Style Sheets) is used to style and layout web pages. It allows you to add colors, fonts, spacing, and much more to your website.

Example: Changing Text Color

h1 {
  color: blue;
}

Selectors in CSS

  • Element Selector: Targets all elements of a specific type. Example: p { color: red; }
  • Class Selector: Targets elements with a specific class. Example: .myClass { font-size: 20px; }
  • ID Selector: Targets an element with a specific ID. Example: #myId { background-color: yellow; }

Example: Adding a Background Color

body {
  background-color: lightgray;
}

Box Model

The box model describes how elements are displayed on a web page. It includes margins, borders, padding, and the actual content.

div {
  width: 300px;
  padding: 20px;
  border: 5px solid black;
  margin: 15px;
}

Fonts and Text Styling

CSS allows you to change fonts and style text easily.

h1 {
  font-family: 'Arial', sans-serif;
  font-size: 24px;
  text-align: center;
}

JavaScript Basics

JavaScript is the programming language that makes web pages interactive. It allows you to manipulate HTML elements, respond to user actions, and dynamically update content on the page.

What is JavaScript?

JavaScript is a lightweight, interpreted programming language primarily used for adding interactivity to websites. It works alongside HTML and CSS to create dynamic content such as form validation, animations, and interactive maps.

Variables

In JavaScript, variables are used to store data. You can declare variables using the let, const, or var keywords.

let name = "John";
const age = 30;
var city = "New York";

Functions

Functions in JavaScript are blocks of code designed to perform a particular task. You can call a function to execute it whenever needed.

function greet(name) { return "Hello, " + name + "!"; } alert(greet("Alice"));

Conditionals

Conditionals allow you to execute code based on specific conditions using if, else if, and else.

let age = 18;
if (age >= 18) {
  console.log("You are an adult.");
} else {
  console.log("You are a minor.");
}

Loops

Loops are used to repeat a block of code multiple times. Here’s an example using a for loop:

for (let i = 0; i < 5; i++) { console.log("This is iteration number " + i); }

JavaScript in HTML

JavaScript can be added directly to an HTML page inside a <script> tag. It can either be placed in the <head> or just before the closing </body> tag for optimal performance.
